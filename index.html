什么是强化学习？

强化学习（Reinforcement Learning, RL）是一种机器学习范式，其中智能体（Agent）通过与环境（Environment）的交互来学习最优策略（Policy）。智能体在每个时间步观察当前状态（State），选择一个动作（Action），然后接收来自环境的奖励（Reward）并转移到新状态。

马尔可夫决策过程 (MDP)

强化学习问题通常被建模为马尔可夫决策过程：

G = (S, A, P, R, γ)

  • S: 状态空间
  • A: 动作空间
  • P(s'|s,a): 状态转移概率
  • R(s,a,s'): 奖励函数
  • γ: 折扣因子 (0 ≤ γ ≤ 1)

核心概念

🎯 智能体 (Agent)

学习者和决策者，负责选择动作以最大化累积奖励。

🌍 环境 (Environment)

智能体交互的外部世界，提供状态和奖励反馈。

📊 策略 (Policy)

π(a|s) - 在给定状态下选择动作的概率分布。

💰 价值函数 (Value Function)

V^π(s) - 从状态s开始，遵循策略π的期望累积奖励。

⚖️ Q函数 (Q-Function)

Q^π(s,a) - 在状态s执行动作a后，遵循策略π的期望累积奖励。

经典强化学习算法

1. Q-Learning (表格型)

一种无模型的时序差分学习算法，直接学习最优Q值函数。

// Q-Learning 更新规则
Q(s,a) ← Q(s,a) + α[r + γ·maxₐ' Q(s',a') - Q(s,a)]

特点：离策略学习，适用于离散状态-动作空间

2. Deep Q-Networks (DQN)

使用深度神经网络近似Q函数，结合经验回放和目标网络稳定训练。

// DQN 损失函数
L(θ) = E[(r + γ·maxₐ' Q(s',a';θ⁻) - Q(s,a;θ))²]

关键技术：经验回放、目标网络、ε-greedy探索

3. Policy Gradient

直接优化策略参数，通过梯度上升最大化期望回报。

// REINFORCE 算法
∇J(θ) = E[∇log π(a|s;θ) · Gₜ]

优势：适用于连续动作空间，端到端学习

4. Actor-Critic

结合值函数方法和策略梯度方法，Actor学习策略，Critic评估动作价值。

// Advantage Actor-Critic
A(s,a) = Q(s,a) - V(s)
Actor: ∇J(θ) = E[∇log π(a|s;θ) · A(s,a)]
Critic: TD-error = r + γV(s') - V(s)

5. Proximal Policy Optimization (PPO)
